# Acelera 1.0 by Pedro Santana

Connect an Arduino to the ADXL345 breakout board.

```
#############################
# Arduino PIN # ADXL345 Pin #
#############################
#    10       #     CS      #
#    11       #     SDA     #
#    12       #     SDO     #
#    13       #     SCL     #
#    3V3      #     VCC     #
#    Gnd      #     GND     #
#############################
```
